# Godout
A Godout é uma plataforma de planejamento e gerenciamento de projetos múltiplos, que permite o gerenciamento de funções, equipes e a criação de documentos necessários do projeto. visando conter funcionalidades como quadro Kanban, diagramas de classe, atividade, caso de uso documentação (gdd ABNT, plano de projeto), ciclo de vida e gerenciamento inteligente.
